ezdlin
======

EZD device programmer for Linux, to be used with the devices: zl5prog
(http://www.btc.pl/pdf/zl5prg.pdf) and zl6prog (http://www.btc.pl/pdf/zl6prg.pdf). it is Linux
equivalent of Easy-Downloader (http://www.kmitl.ac.th/~kswichit/#8051), for Windows.

it is tested with the following microcontrollers types:
AT89C51
AT89C2051
